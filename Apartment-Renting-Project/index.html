Apartment Renting Website

This is a website i'm working on for an holidays apartment in Lido Di Pomposa.

Colors:

Pages: 2 (Available in 3 languages)

Made with: Wordpress

Structure:

It is structured into 2 pages. One is the main landing page, where you can find all you need to know about the apartment and contact the owner for any question using the form. The second page is where you can book your vacation using the calendar of the page. Once the period of stay has been chosen, the customer can see a preview of the price. After completing the form, the customer is able to pay with Stripe or Paypal and get their reservation done.

This website is still under development.
